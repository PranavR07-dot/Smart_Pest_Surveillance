🌾 Smart Pest Surveillance and Pesticide Recommendation System

The Smart Pest Surveillance System is an AI and IoT-based solution designed to monitor farm environmental conditions and provide pest risk prediction along with pesticide recommendations.

The system analyzes soil moisture, temperature, humidity, and crop type to identify potential pest threats and suggest suitable pesticides for effective pest control.

This helps farmers make data-driven decisions, reduce crop loss, and improve agricultural productivity.

📌 Project Overview

Pest infestations are one of the major reasons for crop damage in agriculture. Traditional pest monitoring methods are manual and often detect problems too late.

This system provides a smart farming solution by:

Monitoring environmental conditions

Predicting pest risk

Recommending pesticides based on crop and environmental parameters

The system uses Machine Learning and IoT sensor data to assist farmers in early pest detection and management.

🎯 Objectives

Monitor farm conditions using IoT sensors

Analyze environmental data (temperature, humidity, soil moisture)

Predict possible pest infestations

Recommend pesticides based on crop type and conditions

Provide a user-friendly dashboard for farmers

⚙️ System Working

1️⃣ IoT sensors collect environmental data from the farm.

2️⃣ The system receives data such as:

Soil Moisture

Temperature

Humidity

Crop Type

3️⃣ The Machine Learning model analyzes the conditions.

4️⃣ Based on the analysis, the system:

Predicts possible pest risk

Suggests suitable pesticides for the selected crop

5️⃣ The results are displayed on an interactive Streamlit dashboard.

▶️ Installation

Clone the repository

git clone https://github.com/yourusername/smart-pest-surveillance-system.git

Move to project directory

cd smart-pest-surveillance-system

Install required libraries

pip install -r requirements.txt
🚀 Run the Application

Run the Streamlit dashboard

streamlit run streamlit_app.py

The dashboard will open in your browser.
